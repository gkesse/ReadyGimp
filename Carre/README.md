##### Créer Carré
---

* Créer Calque
    * Calque
    * Nouvelle Calque
    * Nom du Calque
    * Carre
    * Valier

* Créer Carré
    * Sélectionner
    * CALQUE_CARRE
    * Outil de Sélection Rectangulaire
    * Activer
    * Lissage
    * Désactiver
    * Fixe
    * Cliquer/Glisser Dessus
    * Pour Dessiner Carré
