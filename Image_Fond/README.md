# Procédure Suivie
---
### Créer Image de Fond

* Créer Nouvelle Image
    * Fichier
    * Nouvelle Image

* Définir une Couleur
    * Outil Couleur Premier Plan
    * Notation HTML
    * 051039
    * Valider

* Ajouter une Couleur
    * Outil Remplissage
    * Mode
    * Normal
    * Opacité
    * 100
    * Sélectionner
    * CALQUE_ARRIERE_PLAN
    * Cliquer Dessus
    * Pour Appliquer Couleur
