# Procédure Suivie
---
### Créer Ellipse

* Créer Calque
    * Calque
    * Nouvelle Calque
    * Nom du Calque
    * Carre
    * Valier

* Créer Ellipse
    * Sélectionner
    * CALQUE_ELLIPSE
    * Outil de Sélection Elliptique
    * Activer
    * Lissage
    * Désactiver
    * Fixe
    * Cliquer/Glisser Dessus
    * Pour Dessiner Ellipse

* Définir une Couleur
    * Outil Couleur Premier Plan
    * Notation HTML
    * 373f60
    * Valider

* Ajouter une Couleur
    * Outil Remplissage
    * Mode
    * Normal
    * Opacité
    * 100
    * Sélectionner
    * CALQUE_ELLIPSE
    * Cliquer Dessus
    * Pour Appliquer Couleur
