# Procédure Suivie
---
### Créer Carré

* Créer Calque
    * Calque
    * Nouvelle Calque
    * Nom du Calque
    * Carre
    * Valier

* Créer Carré
    * Sélectionner
    * CALQUE_CARRE
    * Outil de Sélection Rectangulaire
    * Activer
    * Lissage
    * Activer
    * Fixe
    * Cliquer/Glisser Dessus
    * Pour Dessiner Carré

* Définir une Couleur
    * Outil Couleur Premier Plan
    * Notation HTML
    * 373f60
    * Valider

* Ajouter une Couleur
    * Outil Remplissage
    * Mode
    * Normal
    * Opacité
    * 100
    * Sélectionner
    * CALQUE_CARRE
    * Cliquer Dessus
    * Pour Appliquer Couleur
