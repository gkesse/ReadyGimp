# Procédure Suivie
---
### Créer Image de Fond

* Créer Nouvelle Image
    * Fichier
    * Nouvelle Image

* Définir une Couleur
    * Outil Couleur Premier Plan
    * Notation HTML
    * 051039
    * Valider

* Ajouter une Couleur
    * Outil Remplissage
    * Mode
    * Normal
    * Opacité
    * 100
    * Sélectionner
    * CALQUE_ARRIERE_PLAN
    * Cliquer Dessus
    * Pour Appliquer Couleur
---
### Créer Rectangle

* Créer Calque
    * Calque
    * Nouvelle Calque
    * Nom du Calque
    * Carre
    * Valier

* Créer Rectangle
    * Sélectionner
    * CALQUE_CARRE
    * Outil de Sélection Rectangulaire
    * Activer
    * Lissage
    * Désactiver
    * Fixe
    * Cliquer/Glisser Dessus
    * Pour Dessiner Rectangle

* Permuter Couleur Premier Arrière Plan
    * Outil Permuter Couleur Premier Arrière Plan

* Définir une Couleur
    * Outil Couleur Premier Plan
    * Notation HTML
    * 373f60
    * Valider

* Ajouter une Couleur
    * Outil Remplissage
    * Mode
    * Normal
    * Opacité
    * 100
    * Sélectionner
    * CALQUE_CARRE
    * Cliquer Dessus
    * Pour Appliquer Couleur
